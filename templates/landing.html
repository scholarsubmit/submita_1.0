FAQ

Frequently Asked Questions

Got questions? We've got answers

What is Submita and who is it for?
Submita is an all-in-one assignment management platform designed specifically for Computer Science departments. It serves students, lecturers, and administrators by streamlining assignment submission, plagiarism detection, automated grading, and feedback delivery.
How accurate is the plagiarism detection?
Our plagiarism detection system has a 99% accuracy rate. It uses multi-layer similarity analysis to compare submissions against current and past assignments, as well as external sources. The system provides detailed reports with side-by-side comparisons.
What file formats are supported for submissions?
Submita supports a wide range of file formats including Python (.py), Java (.java), C++ (.cpp), JavaScript (.js), ZIP archives, PDFs, Word documents (.doc, .docx), Excel files (.xls, .xlsx), PowerPoint presentations (.ppt, .pptx), and image files (.jpg, .png, .gif).
Can lecturers customize the grading criteria?
Yes, lecturers have full control over grading criteria. They can set custom rubrics, define test cases for auto-grading, adjust grade weights, and override any auto-generated grades. The system also allows for manual feedback and comments.
Is my data secure on Submita?
Absolutely! Submita uses bank-grade encryption for all data transmission and storage. We implement role-based access control, secure file storage, and regular security audits. Your academic data is protected and never shared with third parties.
How much does Submita cost?
Submita is completely free for educational institutions and departments. There's no credit card required to get started. We believe in making quality education tools accessible to everyone. Contact us for enterprise-level features and support.
